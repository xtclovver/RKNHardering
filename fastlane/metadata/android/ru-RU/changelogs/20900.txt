Добавлено шесть новых детекторов VPN/прокси: расширенные паттерны интерфейсов (utun, ZeroTier, Tailscale, IPsec/xfrm, GRE, L2TP), определение типа VPN-транспорта, TUN/TAP по типу из sysfs, эвристика host-route /32, сканер REST API Clash/mihomo/sing-box и проба слабых учётных данных локального SOCKS5. Два последних сканера — опциональные, включаются в настройках.
2. Добавлено определение эмулятора (QEMU, Genymotion, BlueStacks) и изолированного контекста (вторичный пользователь, клон приложения, рабочий профиль). В этих случаях сетевые тесты ненадёжны, поэтому выдаётся вердикт «нужна проверка», а не «обнаружено».
3. В маркетплейс добавлены три профиля: Stealth Check (только локальные проверки без сети), MTProxy/TG STUN Check (трафик Telegram) и Extended Test (максимальная чувствительность).
4. Каталог маркетплейса теперь проверяется подписью Ed25519, добавлен certificate pinning для GitHub CDN.
5. Исправлено: пользовательский GeoIP-провайдер с шаблоном {ip} больше не теряется при сохранении профиля; экспорт отчёта в Markdown полностью локализован (en/fa/zh); устранено состояние гонки при отмене сканирования.
