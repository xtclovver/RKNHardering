RKNHardering یک برنامه اندرویدی برای تشخیص استفاده از VPN و پروکسی در دستگاه است. روش‌شناسی RKN (سرویس فدرال نظارت بر ارتباطات روسیه) برای شناسایی ابزارهای دور زدن فیلترینگ را پیاده‌سازی می‌کند.

نه ماژول تشخیص مستقل به صورت موازی اجرا می‌شوند و نتیجه نهایی توسط VerdictEngine محاسبه می‌شود:

- GeoIP — بررسی IP عمومی در برابر پایگاه‌های داده GeoIP و ارائه‌دهندگان سیگنال hosting/proxy
- مقایسه IP — مقایسه پاسخ‌های IP چکرهای RU و غیر-RU
- نشانه‌های مستقیم — NetworkCapabilities، تنظیمات پروکسی سیستم، برنامه‌های VPN نصب‌شده
- نشانه‌های غیرمستقیم — رابط‌های شبکه، جدول مسیریابی، DNS، خروجی dumpsys VPN، پروکسی‌های localhost
- سیگنال‌های موقعیت مکانی — MCC/SIM/cell/Wi-Fi و جستجوی BeaconDB
- بررسی Bypass — اسکن localhost برای پروکسی‌های SOCKS5/HTTP، Xray gRPC API، نشت شبکه زیرین
- CDN Pulling — درخواست‌های HTTPS به endpoint های trace برای تعیین IP عمومی
- Call Transport — بررسی دسترسی به STUN و MTProto برای تشخیص تونل‌سازی ترافیک
- نشانه‌های نیتیو — بررسی‌های JNI روی رابط‌ها، مسیرها، ماکرهای hook در /proc/self/maps و root

نتیجه نهایی (تشخیص داده نشد / نیاز به بررسی / تشخیص داده شد) با ترکیب وزن‌دار نتایج ماژول‌ها محاسبه می‌شود.
