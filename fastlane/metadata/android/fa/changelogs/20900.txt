1. شش آشکارساز جدید VPN/پروکسی اضافه شد: الگوهای گسترده اینترفیس (utun، ZeroTier، Tailscale، IPsec/xfrm، GRE، L2TP)، تشخیص نوع انتقال VPN، TUN/TAP بر اساس نوع sysfs، اکتشاف host-route /32، اسکنر REST API برای Clash/mihomo/sing-box و کاوش اعتبارنامه‌های ضعیف SOCKS5 محلی. دو اسکنر آخر اختیاری هستند و در تنظیمات فعال می‌شوند.
2. تشخیص شبیه‌ساز (QEMU، Genymotion، BlueStacks) و تشخیص محیط ایزوله (کاربر ثانویه، اپلیکیشن کلون‌شده، نمایه کاری) اضافه شد. در این موارد تست‌های شبکه قابل اعتماد نیستند، بنابراین به‌جای «شناسایی شد» نتیجه «نیاز به بررسی» داده می‌شود.
3. سه نمایه جدید در مارکت‌پلیس: Stealth Check (فقط بررسی‌های محلی بدون شبکه)، MTProxy/TG STUN Check (ترافیک تلگرام) و Extended Test (حداکثر حساسیت).
4. کاتالوگ مارکت‌پلیس اکنون با امضای Ed25519 تأیید می‌شود و certificate pinning برای GitHub CDN اضافه شد.
5. رفع اشکال: ارائه‌دهنده سفارشی GeoIP با الگوی {ip} هنگام ذخیره نمایه دیگر حذف نمی‌شود؛ خروجی گزارش Markdown کاملاً بومی‌سازی شد (en/fa/zh)؛ شرایط رقابتی هنگام لغو اسکن برطرف شد.
